# Solar System

An simple animation of the solar system, written in Typescript + WebGL.

## Online Demo

[https://pengfeiw.github.io/solar-system](https://pengfeiw.github.io/solar-system)

## Run

1. git clone https://github.com/pengfeiw/solar-system.git
2. cd solar-system
3. npm install
4. npm run start
